# Lufga Font Files

Please place the following Lufga font files in this directory:

## Required Font Files:

1. **Lufga-Regular.woff2** (font-weight: 400)
2. **Lufga-Regular.woff** (font-weight: 400, fallback)
3. **Lufga-Regular.ttf** (font-weight: 400, fallback)

4. **Lufga-Medium.woff2** (font-weight: 500)
5. **Lufga-Medium.woff** (font-weight: 500, fallback)
6. **Lufga-Medium.ttf** (font-weight: 500, fallback)

7. **Lufga-SemiBold.woff2** (font-weight: 600)
8. **Lufga-SemiBold.woff** (font-weight: 600, fallback)
9. **Lufga-SemiBold.ttf** (font-weight: 600, fallback)

10. **Lufga-Bold.woff2** (font-weight: 700)
11. **Lufga-Bold.woff** (font-weight: 700, fallback)
12. **Lufga-Bold.ttf** (font-weight: 700, fallback)

## Font File Formats Priority:

- **.woff2** (recommended - best compression, modern browsers)
- **.woff** (fallback - good compression, wider browser support)
- **.ttf** (fallback - largest file size, universal support)

## Where to Get Lufga Font:

1. If you have the font files, place them directly in this folder
2. If you need to download them, you can get them from:
   - The font's official website or distributor
   - Your design team or font license provider
   - Font repositories (ensure you have proper licensing)

## After Adding Font Files:

Once you've placed the font files here, the application will automatically use them. 
The fonts are configured in `src/index.css` using @font-face declarations.
